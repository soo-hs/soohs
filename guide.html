01
OVERVIEW
システム概要
📋
2,949件の求人情報
鹿児島・宮崎をはじめ全国の求人を一覧管理。すべてのデータが1画面で確認できます。
🔍
多彩な絞り込み機能
求人番号・事業所名・業種・産業分類・年間休日数など8つの条件で瞬時に絞り込めます。
📱
PC・スマホ両対応
ボタン一つでスマートフォン表示に切り替え。校内どこからでも快適に閲覧できます。
💡
ファイルについて
受付求人票は 曽於高校_受付求人票.html という単一のHTMLファイルです。インターネット接続不要で動作します。ブラウザで直接開くだけで使用できます。
02
BASIC OPERATION
基本操作の手順
01
🌐
ファイルをブラウザで開く
▼

HTMLファイルをダブルクリックするか、ブラウザ（Chrome / Edge / Safari）にドラッグ＆ドロップして開きます。

  • 推奨ブラウザ：Google Chrome / Microsoft Edge / Safari
  • Internet Explorer は非対応です
⚠ 初回読み込み時、データの展開に数秒かかる場合があります。「データを読み込み中…」の表示が消えるまでお待ちください。
02
🖥
画面構成を確認する
▼

画面は上から順に次の4エリアで構成されています。

  • ヘッダー（固定） — タイトル・全件数・絞り込みボタン・スマホ切替ボタン
  • 検索パネル（開閉式） — 各種絞り込み条件の入力欄
  • カラムヘッダー（固定） — 各列の見出し。クリックでソート
  • 一覧テーブル（スクロール） — 求人データ本体
  • フッター（固定） — ページネーションコントロール
03
📜
一覧を閲覧する
▼

テーブルエリアを上下にスクロールして求人情報を確認します。左右にスクロールすると隠れている列（資本金・賞与・業種など）も表示されます。

  • 1ページあたり 50件 表示されます
  • 初期表示は 独自番号の昇順 でソートされています
  • カラムヘッダーは常に画面上部に固定されています
03
SEARCH & FILTER
検索・絞り込み機能
STEP
🔍
検索パネルを開く
▼

ヘッダー右側の 🔍 絞り込み ボタンをクリックすると検索パネルが表示されます。もう一度クリックすると閉じます。

検索パネル右下に 絞り込み結果件数 がリアルタイムで表示されます。

🔤 求人番号 前方一致
入力した文字列で始まる求人番号を検索します。都道府県コードで絞り込む際に便利です。
例: 46080 → 大隅管轄の求人のみ表示
🏢 事業所名 曖昧検索
会社名の一部を入力するだけで部分一致で検索できます。株式会社の省略も可能です。
例: ナンチク → 「ナンチク」を含む全事業所
📍 所在地 曖昧検索
事業所の住所（所在地）で絞り込みます。都道府県名・市区町村名で検索できます。
例: 曽於市 → 曽於市内の事業所のみ
🗺 就業場所 曖昧検索
実際に働く場所（就業場所）で絞り込みます。所在地と異なる場合の確認に使います。
例: 鹿児島県 → 就業場所が鹿児島県の求人
👷 職種 曖昧検索
職種名の一部で絞り込みます。希望する業務内容のキーワードで検索するのが効果的です。
例: 事務 → 「事務」を含む職種すべて
🏭 業種8種 プルダウン
8つの大分類からプルダウンで選択します。製造・建築・販売・運輸・飲食・医療・理美容・その他。
例: 医療・福祉 → 病院・介護施設の求人のみ
📊 産業分類 プルダウン
239種類の詳細業種からプルダウンで選択します。業種8種より細かい条件で絞り込めます。
例: 畜産食料品製造業 → 食肉関連工場のみ
🗓 年間休日数 範囲指定
最小・最大の日数を数値で入力して範囲指定します。片方だけ入力することも可能です。
例: 120〜 → 年間休日120日以上の求人のみ
🔗
複数条件の組み合わせ
すべての検索条件はAND検索で機能します。複数の条件を同時に入力することで、より精度の高い絞り込みが可能です。
リセットボタンをクリックすると全条件を一括でクリアできます。

列幅の変更
各カラムヘッダーの右端にマウスカーソルを合わせるとリサイズカーソルが表示されます。そのままドラッグすることで列幅を自由に調整できます。
05
SORT
並び替え（ソート）
HOW
↕️
カラムヘッダーをクリックしてソート
▼

画面上部に固定されているカラムヘッダーの任意の列名をクリックすると、その列を基準にデータが並び替えられます。

  • 1回クリック → 昇順（▲）でソート
  • 2回クリック → 降順（▼）でソート
  • 現在のソート列は ▲ / ▼ のアイコンで表示されます
  • 数値列（月額賃金・求人数・年間休日数など）は数値順でソートされます
  • 文字列列は日本語の辞書順でソートされます
💡 月額賃金で降順ソートすると、給与の高い求人から順に確認できます。年間休日数で降順ソートすると、休日の多い企業が上位に並びます。

07
PAGINATION
ページネーション
INFO
📄
ページの操作方法
▼

画面下部のフッターに固定されたページネーションコントロールでページを切り替えます。

  • « — 最初のページへ移動
  • ‹ — 前のページへ移動
  • 数字ボタン — 任意のページへ直接移動（前後3ページ分のボタンが表示）
  • › — 次のページへ移動
  • » — 最後のページへ移動
  • 右側の XX–YY / ZZZ件 で現在表示中の範囲と総件数が確認できます
ℹ 1ページあたり50件表示されます。絞り込みを行うと自動的にページ1に戻ります。
08
SMARTPHONE
スマートフォン表示
STEP
📱
スマホモードへの切り替え
▼

ヘッダー右端の 📱 スマホ ボタンをクリックすると、スマートフォン向けの表示モードに切り替わります。

  • スマホモードではページ全体がスクロール可能になります
  • カラムヘッダーは画面上部にスティッキー固定されます
  • テーブルは横スクロールで全列を確認できます
  • カラムヘッダーと各セルの幅は常に同期されています
  • 再度ボタン（🖥 PC）をクリックするとPCモードに戻ります
💡 実際のスマートフォンでファイルを開く際もスマホモードボタンをタップすることで快適に閲覧できます。
⚡
操作のコツまとめ
① まず業種8種プルダウンで大まかに絞り込む
② 次に職種のキーワードで候補を絞る
③ 月額賃金または年間休日数で降順ソートして比較
④ 列幅をドラッグ調整して読みやすくカスタマイズ
⑤ 詳細が必要な求人は求人番号を控えてハローワークで確認
